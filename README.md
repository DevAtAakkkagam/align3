# align3

Terni Lapilli (three men's morris) — 2-player local hotseat.

**Play: [games.aakkagam.com/align3](https://games.aakkagam.com/align3)**

Two board variants:

- **Roman circle** — win on 4 spokes or any 3 consecutive rim points
- **Tamil square** — win on the 8 straight lines of a 3×3 grid

## Stack

Svelte 5 + TypeScript + Vite. No backend.

## Develop

```sh
npm install
npm run dev     # dev server
npm test        # tests
npm run build   # build to dist/
```
